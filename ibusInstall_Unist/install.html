Ubuntu 18.04 安装中文输入法ibus

分步阅读

在安装ubuntu时，如果选择英文安装默认会不启用中文输入法的。可以通过下述方法开启

步骤阅读

工具/原料

  • Ubuntu 18.04

方法/步骤

  1. 1

    安装 pingyin 输入法

    sudo apt-get install ibus-pinyin

  2. 2

    然后在 settings 的 Region & Language 的 Input Sources设置栏中

    点击 Manage     Installed Language ，初次进入会安装些字体等相关信息。注销后使之生效。

    步骤阅读
  3. 3

    点击 + 添加Chinese( Pinyin)。注销后使之生效。
